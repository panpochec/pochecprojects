GaussianMaster Python Package

While working on the Schiff bases project I have found sigificant obstacles in developing a clean, well thought out code when working under time constrains. After all, computational chemistry is subjected not only to our own time, but also timetables and resources provided by supercompuitng centers. And so GaussianMaster was born.

It encompasses most of the functions to extract typical and more exotic data from Gaussian logfiles, such as atom matrix, distance, angle and dihedral angle matrix, cleaning of that data (by turning atom numbers into atom type labels), extracting bridge locations and many more. It can also create custimizable input files for different applications. Finally, it can read energy profiles from scans.

All of this is achieved using Pandas and NumPy packages and is rather efficient when it comes to resources.

The package is well documented, both within the scripts (comprehensive docstrings) as well as on GitHub repository - with pdf containing documentation/tutorial.

I'm planning to further develop the package by adding more functionalities - both for my own needs, but also per users requests. To learn more abut GaussianMaster, read the code and documentation and maybe even give it a try - visit its GitHub Repository or PyPI page.

Oh, and you can even get it using PIP! So pip install gaussianmaster!
